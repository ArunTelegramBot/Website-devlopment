# Project Rules
1. Only work on the task I explicitly give.
2. Do not create extra features unless asked.
3. Do not touch unrelated files.
4. Do not search the web.
5. Prefer the smallest possible change.
6. Before changing code, explain plan in 3-5 lines.
7. If unclear, ask ONE question only.
8. After work, return only: files changed, what changed, how to test, risks.
9. Never rewrite whole project unless I request it.
10. Stop immediately after task is done.
